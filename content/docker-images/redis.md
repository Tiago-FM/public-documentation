## Image

Image is named `fpfis/redis` and based on CentOS 6.

## Configuration

No configuration is available for this image.

## Port

Redis is running on port `6379` by default.

## Example

```bash
docker run -p 6379:6379 -ti --rm fpfis/redis
```
